# Quantium Virtual Internship — Chips Category Analytics

Data analytics case study completed as part of Quantium's Retail Analytics 
Virtual Experience Program (via Forage).

## 📁 Project Structure
- `notebooks/` — Python analysis notebooks for Task 1 and Task 2
- `data/` — raw input datasets
- `charts/` — all generated visualisations
- `report/` — final PDF submissions and client presentation

## Task 1 — Data Preparation & Customer Analytics
- Cleaned and merged transaction and customer datasets (removed duplicates, 
  non-chip products, and an outlier bulk-purchase transaction)
- Derived pack size and brand features from raw product names
- Identified key customer segments driving category sales — Mainstream 
  Young Singles/Couples and Mainstream Retirees emerged as priority segments
- Delivered brand and pack-size recommendations for each target segment

## Task 2 — Experimentation & Uplift Testing
- Selected control stores for 3 trial stores using a combined correlation 
  + magnitude distance similarity score, validated visually
- Assessed trial store performance against matched controls using a 
  statistical significance test (t-test vs pre-trial baseline variation)
- Found a significant, growing sales uplift in 2 of 3 trial stores, driven 
  by increased customer visits

## Task 3 — Client Report
- Summarised findings from Task 1 and 2 into a client-ready presentation 
  using the Pyramid Principle framework

Python (pandas, matplotlib, scipy), Excel, PowerPoint
